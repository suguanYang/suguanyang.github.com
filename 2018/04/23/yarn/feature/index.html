the feature of yarn

yarn 是由facebook的工程师开发的包管理器

快速，稳定，安全依赖管理 是yarn给用户提供的保证，

离线缓存

当你使用yarn安装一个包时(yarn add packageName), 它会将包放置到硬盘中，下一次安装这个包时，yarn不会去请求网络而是直接从注册处那儿取得压缩包，
你的缓存模块会被放置在~/.yarn-cache，将会以注册处名称为前缀，模块版本作为后缀。

确定性安装(Deterministic Installs)

Yarn 使用lockfiles(yarn.lock)和確定性安裝算法，我們可以對“在我的電腦上好好的”這種bug說再見了。

它对所有的依赖项包含了額外的版本號。

许可证检查(License checks)

Yarn 带有一个方便的许可证检查器，当你必须检查所有依赖项的证书时是非常有用的

潜在问题

Yarn 还处在早期阶段， 所以当你在使用时发生一些错误是不必惊讶的。
