http:

HyperText Transfer Protocol

  • the http 是应用层上的协议，应用层可以和基于web的应用进行交流与数据的交换

  • the http 是web的信使

  • http是基于TCP/IP协议的

  • http可以传送内容，像图片，文字，语音，视频等

http的几个重要特征：

  • http is connectionless: 在一次请求结束以后，客户端将会与服务器断开链接，之后，当响应已经准备完毕，服务端将会再次重新启动链接并且传输数据
  • http可以发送任意排序的数据，只要服务端有能力读懂。
  • the http is stateless: 客户端和服务端都知道对方是处于本次请求期间，如果请求被关闭，而且双方想再次链接，它们需要重新提供信息给对方，这个链接将会第一个被处理

why http?

  • http被设计主要用来获取html，将它们发送到客户端
  • http被设计的很精致
  • http将会不断的进化而且更多的特性将会被添加
  • http是最方便达到快速，可靠的在web上移动数据的方式
